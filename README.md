# Amazon-Style Navbar with Sign-In Button

This project is a simple front-end implementation of an **Amazon-inspired navigation bar** with a **Sign In button**, built using only **HTML and CSS**.  
The goal is to practice layout structuring, navigation design, and basic UI styling.

## Live Demo
https://mehedi-x.github.io/amazon-signin-navbar/

## Repository Purpose
- Practice real-world navbar design
- Learn how large websites structure navigation
- Improve CSS positioning and styling skills
- Build a reusable navbar component
